Design Ziel

  • Responsivness - Seite muss breite und schmaler funktionieren als bisher
  • das Programm sollte ähnlich übersichtlich bleiben
  • Das Farbschema muss Jahr für Jahr geändert werden können
  • Es muss ein "Headerbild" (könnten auch 2 sein, gross/klein) geben, welches der Grafiker Jahr für Jahr neu Designen kann
  • Es kann, muss aber nicht unbedingt einen Handgezeichneten Footer geben
  • Wenige andere handgemalte Objekte wären möglich

API

  • festival-jups.ch -> Liveseite
  • admin.festival-jups.ch -> Backend
  • api.festival-jups.ch/events -> Api
    • get api.festival-jups.ch/events - get all events
    • get api.festival-jups.ch/events/:id - get one event
    • post api.festival-jups.ch/events/:id - add one event
    • put api.festival-jups.ch/events/:id - update one event
    • delete api.festival-jups.ch/events/:id - delete one event
  • api.festival-jups.ch/pages -> Api
    • get /pages/home - get content of "home"
    • put /pages/home - update content of "home"
    • get /pages/orte - get content of "orte"
    • put /pages/orte - update content of "orte"
    • get /pages/kontakt - get content of "kontakt"
    • put /pages/kontakt - update content of "kontakt"
    • get /pages/home - get content of "archiv"
    • put /pages/home - update content of "archiv"
  • api.festival-jups.ch/uploads -> Alles Uploads (pds, images, ...)

Felder Events

  • Type ("veranstaltung", "workshop", "offenes angebot")
  • Titel (String)
  • Untertitel (String)
  • Datum (Date)
  • Ort (String)
  • Zeit von (Time)
  • Zeit bis (Time)
  • Priorität (bei gleicher Startzeit) (Number)
  • Bild (quadratisch) (String (pfad))
  • 1 Logo für allfällige Sponsoren (querformatig) (String (pfad))
  • Text (html Möglich) ("string")
  • Ausverkauft (boolean)
  • Ausverkauft-Text (Es gibt noch Stehplätze an der Abendkasse) (String)
  • Optional (wenn es vom Design her Sinn ergibt:
    • Ab Alter ("String")
    • Türöffnung (XY min vor Beginn) ("String")
    • Preis (Frei, keine Angabe, CHF) ("String")

Felder Pages

  • {content: "String mit <b>rohem</b> HTML"}
